Ragnarok Online Character Simulator

WORK IN PROGRESS - BETA

This project is meant to start by simulating a character's skill builds, but will slowly evolve to also encompass statistics, equipment and damage if time and other workloads permit.

Please report bugs, when the project becomes more advanced, and make suggestions about ANYTHING to gui@exodus.io if you want to chat about ideas, provide me links to damage calculations/calculators, or translate the end project to other languages.
